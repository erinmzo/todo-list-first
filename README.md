# Task to Easy
This is an application where you can easily manage tasks.<br/>
You can delete and mark tasks as complete, and track their progress.<br/>
(간편하게 task를 관리할 수 있는 어플리케이션입니다. 삭제 및 완료 처리를 할 수 있고, 진행도를 파악할 수 있습니다.)
